class pipesnake.base.abstract.Transformer(name=None, sklearn_output=False, **kwargs)[source]¶

Bases: sklearn.base.BaseEstimator

Transformer abstract class freely inspired by Scikit-Learn.

See TransformerMixin for more details, the main differences are the parameters type and the returns. Note: that this inherits from sklearn BaseEstimator to simplify the parameters managment.

Parameters:
  • name – provide a name for this transfomer
  • sklearn_output – if True produces outputs compatible with sklearn Pipeline
