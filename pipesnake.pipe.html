pipesnake.pipe.parallel module¶

class pipesnake.pipe.parallel.ParallelPipe(transformers=[], reset_index=True, sklearn_output=False, name=None)[source]¶

Bases: pipesnake.base.abstract.Pipe, pipesnake.base.abstract.Transformer

ParallelPipe applies a list of transformers as parallel.

The class itself inherits from Pipe and Transformer so that it is possible to use this as Transformer objects.

 a list of Transformer objects
